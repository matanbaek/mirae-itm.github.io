Section 3 — AI 코치 파이프라인

GPT-4o에 주 1회 훈련 플래닝 위탁

Garmin FIT → Python 파싱 → JSON → GPT-4o 프롬프트 → 다음 주 플랜까지의 파이프라인.

파이프라인

1. 수집Garmin Connect API · FIT 파일
2. 파싱Python fitparse → JSON
3. 집계주간 총량·심박 존·페이스 구간
4. 프롬프팅GPT-4o + 훈련 이론 시스템 프롬프트
5. 검토본인 컨디션 메모로 조정
6. 실행다음 주 캘린더 자동 기입

시스템 프롬프트에는 Jack Daniels의 VDOT 표, 80/20 분배(Seiler), 회복 이론이 압축되어 있습니다. 모델은 단순 답변 대신 구체적 페이스·km·인터벌 세트를 제시하도록 제약됩니다.
